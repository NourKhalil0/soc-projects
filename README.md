## 🗺️ Roadmap

Projects will cover these core SOC skill areas:

- **Log Analysis** — auth logs, firewall logs, Windows event logs
- **Network Monitoring** — port scanning, traffic analysis, DNS inspection
- **Threat Detection** — IOC extraction, anomaly detection, alert triage
- **Incident Response** — file integrity, process monitoring, artifact collection
- **OSINT & Threat Intel** — IP reputation, hash lookup, domain analysis
- **SIEM & Automation** — alert parsing, rule building, report generation
